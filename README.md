# Stock Alerts
Flutter app that will show/alert details about realtime stock changes in US stock market.
#### Features
- Log In / Log Out
- Sign Up
- Portfolio
- Chat
- Stock Signals
- Watchlist
- Search for stock
- View details about specific stock
- Stock Graphs
- Stock News

#### Tech
- Flutter
- Dart
- Firestore
- Firebase cloud messaging
- Note Js / Express Js
- Heroku
- IEX Cloud
